# LumaLab3D — Jekyll Site

A colorful premium landing site that redirects shoppers to Etsy for now.

## Quick Start
1. Create a new repo (e.g., `lumalab3d.github.io`).
2. Copy this structure into it.
3. Place your banner in `assets/img/banner.png`.
4. Enable GitHub Pages → Build from branch.
5. Update `_config.yml` with your URL.
6. Optional: Add a CNAME file with your domain (e.g., `LumaLab3D.com`).
